# ♟️ Xadrez Online | Studio de IA

Este projeto é um jogo de xadrez moderno e responsivo, desenvolvido inteiramente através de uma colaboração entre humanos e diversos Agentes de IA da equipe **BMAD**. O objetivo foi criar uma experiência premium, educativa e funcional, pronta para ser hospedada gratuitamente no GitHub Pages.

## 🚀 Como Jogar

1.  **Acesse o link:** (Insira sua URL do GitHub Pages após o deploy).
2.  **Escolha o Modo:**
    *   **Contra Amigo:** Gere seu ID P2P e envie para um amigo se conectar.
    *   **Contra I.A.:** Desafie o motor Stockfish em 4 níveis (Fácil ao Lendário).
3.  **Movimente as Peças:** Clique e arraste ou clique na peça e depois na casa de destino.
4.  **Aprenda:** Se você estiver no modo **Fácil** e entrar em **Xeque**, o assistente sugerirá uma jogada de defesa!
5.  **Tutorial:** Clique no ícone `❓` a qualquer momento para ver as regras básicas.

## 🛠️ Tecnologias Utilizadas

*   **Chess.js**: Mecanismo de regras e validação.
*   **Chessboard.js**: Renderização visual do tabuleiro.
*   **Stockfish.js**: I.A. de alto nível rodando via Web Workers.
*   **PeerJS**: Conexão Multiplayer P2P (sem necessidade de servidor).
*   **Vanilla CSS**: Design fluido com vmin e Glassmorphism.
